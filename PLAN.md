# Last Light — build plan

A real **Godot 4.6.3** third-person survival-horror, exported to HTML5/WebGL2 (gl_compatibility,
nothreads) for mobile browsers. **Not** a web/JS/Canvas app.

See the PR description for the full design. Branch builds the game described in the request:
keep a campfire alive at an abandoned rural bus stop, scavenge wood, fight skeletons that rise
from the dark with melee, survive a photographic dusk-to-night to dawn.
